# Task Package: ORC-05 Major Distribution Structured Executor

## Source Context

- `docs/retrieval_recall_v1_implementation_plan.md`: ORC-05 adds the first structured SQL executor for `major_distribution`.
- `nexus_app.retrieval.domain_registry`: registered `major_distribution` query profiles and field whitelists.
- `nexus_app.retrieval.schemas`: structured plan/result/source-ref contracts.
- `nexus_app.models.MajorDistributionDataset` and `MajorDistributionRecord`: Pipeline B major distribution domain tables.

## Goal

Add a guarded structured retrieval executor for `major_distribution` that validates LLM-produced structured plans against query profiles and field whitelists, then executes safe SQLAlchemy queries.

## Scope

- Add reusable structured SQL guardrail validation.
- Add `MajorDistributionRetrievalExecutor`.
- Support first-batch query profiles:
  - `major_distribution.trend_by_year`
  - `major_distribution.by_province`
  - `major_distribution.by_education_level`
  - `major_distribution.record_list`
- Add focused tests for guardrails and executor behavior.

## Out Of Scope

- `job_demand` executor.
- `competency_analysis` executor.
- API or Console integration.
- Orchestrator integration.
- Natural-language-to-SQL execution.
- Permission or L3/L4 filtering.

## Forbidden Changes

- Do not execute raw SQL generated by LLM.
- Do not allow fields outside the registered query profile whitelist.
- Do not introduce RAGFlow or Evidence Graph as retrieval sources.
- Do not change existing `/open/v1/major-distribution-*` APIs.
- Do not add database migrations.

## Deliverables

- `nexus-app/nexus_app/retrieval/sql_guardrails.py`
- `nexus-app/nexus_app/retrieval/executors/major_distribution.py`
- `nexus-app/tests/retrieval/test_sql_guardrails.py`
- `nexus-app/tests/retrieval/test_major_distribution_executor.py`

## Acceptance

- Valid `major_distribution.trend_by_year` plans pass guardrail validation.
- Unknown query profile, non-whitelisted filter, non-whitelisted order field, and over-limit plans fail closed.
- The executor returns year aggregation for a three-year major distribution query.
- The executor returns record-list results with source refs and row locators.
- Tests use SQLite fixtures and do not perform network calls.
